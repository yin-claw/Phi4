# Frontier Obligations Report

This directory contains machine-readable inventories of explicit frontier debt.

- `frontier.tsv` is generated by:
  - `scripts/frontier_report.sh --emit docs/frontier_obligations/frontier.tsv`
- It is also refreshed by:
  - `scripts/check_phi4_trust.sh`
  - `scripts/weekly_gate.sh`

`frontier.tsv` rows use this schema:

- `kind`: one of `gap_theorem`, `model_class`, `nonempty_constructor`
- `name`: declaration name
- `file`: source file path
- `line`: 1-based declaration line

This report exists to make frontier obligations externally auditable even when
frontier staging uses assumption-explicit `Nonempty (...)` patterns.
